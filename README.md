# GraphiChat

Simple chat app for learning/testing GraphQL API with the following stack:

Server:
 -Node/Express
 -Apollo Server
 -GraphQL Modules
DB:
 -MongoDB
Client:
 -React
 -Apollo Client/React Apollo
 -Webpack
UI:
 -MaterialUI
 -Styled-Components
Testing:
 -Jest
 -Enzyme
Other features:
 -OAuth with refresh token
 -Social login/registration
 -List with custom scrollbar and lazy loading
 -React Hooks
 -Docker
